## 2024-05-24 - Add tooltip and click handler for Export MIDI button
**Learning:** Found that the Textual `Button` class supports a `tooltip` attribute natively, which is an excellent way to expose keyboard shortcuts (like `[E]`) directly on buttons without cluttering the UI. Also found that `Button` clicks must be wired manually using `on_button_pressed(self, event: Button.Pressed)` in the Textual App if they don't use the native `action` attribute string mapping (e.g., `action="export_midi"` would also work if the button wasn't dynamically yielding, but explicitly capturing the click by id ensures deterministic behavior).
**Action:** Next time I encounter a Textual TUI with keybindings, I will check if the corresponding interactive buttons have tooltips documenting the shortcut, and ensure they have a corresponding click handler.
